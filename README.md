# dontrust

Don't trust, verify.

A **highly visible** CLI tool for managing multisig on Aptos.

## Development

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Build: `pnpm build`
4. Link locally: `pnpm link`
5. Format code: `pnpm format`
6. Run in dev mode: `pnpm start`
